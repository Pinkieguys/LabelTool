# Contributing to LabelTool

First off, thank you for considering contributing to LabelTool! It's people like you that make LabelTool such a great tool.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub issue tracker.
- Describe the bug and include steps to reproduce it.

### Suggesting Enhancements
- Open an issue to discuss your idea.

### Pull Requests
1. Fork the repo.
2. Create a new branch.
3. Make your changes.
4. Add tests if possible.
5. Submit a pull request.

## Development Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/labeltool.git
   cd labeltool
   ```
2. Create a virtual environment and install dependencies:
   ```bash
   python -m venv venv
   source venv/bin/activate  # or venv\Scripts\activate on Windows
   pip install -e .
   ```

## Code Style
- Follow PEP 8.
- Use clear variable names.
- Document functions and classes.
